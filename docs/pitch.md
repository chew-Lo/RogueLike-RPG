# Pitch
**Track:** Games / Entertainment
**Product:** RogueLike RPG - A browser-based strategic combat game with class selection and skill progression.

**Problem & User:** Casual gamers who enjoys rogue like game. Players want something simple quick to access and enjoyable game experience.

**Core Loop:** Players select a character class (Warrior, Archer, Wizard) and enter automated battles against increasingly difficult enemies. After each victory, they choose from randomized rewards including stat upgrades, new skills, or healing. The game tracks high scores and wave progression locally while synchronizing leaderboard data to the cloud. Combat is automatic but strategic through class selection, skill unlocks, and upgrade choices that permanently enhance capabilities across runs.
